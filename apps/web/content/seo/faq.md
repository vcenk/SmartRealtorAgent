## Getting Started

### How quickly can I get started?

Most users deploy their AI assistant within an hour. The process involves:
1. Creating an account (2 minutes)
2. Uploading your content (10-30 minutes)
3. Customizing your widget (5 minutes)
4. Embedding the script (2 minutes)

No technical expertise required.

### Do I need a developer to install the chatbot?

No. Installation requires adding a single script tag to your website. If you can edit your website's HTML or use a tag manager, you can install Smart Realtor Agent yourself.

### What websites does it work with?

Smart Realtor Agent works with any website platform:
- WordPress
- Squarespace
- Wix
- Webflow
- IDX websites
- Custom-built sites
- Any platform that allows script tags

### Can I try it before committing?

Yes. We offer a free tier so you can test the platform with your actual content before upgrading.

## Features & Functionality

### How does the AI know about my listings?

You upload your content to the knowledge base. This can include:
- Website pages (via web crawler)
- PDF documents
- Text files
- Direct URL imports

The AI only answers based on information you provide, reducing hallucinations.

### Will the chatbot make up information?

Smart Realtor Agent uses RAG (Retrieval-Augmented Generation) architecture. This means:
- Responses are grounded in your uploaded content
- Citations show where information comes from
- The AI admits when it doesn't know something
- Hallucination risk is significantly reduced compared to generic chatbots

### Can it qualify leads automatically?

Yes. The chatbot can:
- Detect buyer vs. seller intent
- Ask qualifying questions naturally
- Capture budget, timeline, and location preferences
- Store structured lead data in your dashboard

### Does it work 24/7?

Yes. Once deployed, the chatbot responds instantly at any time of day or night. No human monitoring required.

## Security & Privacy

### Is my data secure?

Yes. We implement multiple security layers:
- Row-level security in the database
- Tenant isolation for all data
- Encrypted storage
- Secure authentication
- No cross-tenant data access

### Is my data shared with other users?

No. Each account (tenant) has completely isolated data. Your knowledge base, leads, and conversations are never visible to other users.

### Is my content used to train AI models?

No. Your uploaded content is used only to generate responses for your chatbot. It is not used to train any external AI models.

### Are you GDPR compliant?

We implement data protection measures aligned with GDPR principles, including data isolation, user consent mechanisms, and data deletion capabilities.
